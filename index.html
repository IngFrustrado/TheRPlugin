R plugin

The R plugin for PyCharm and IntelliJ provides smart editing for R scripts. It's open source and cross-platform.

It includes syntax-highlighting editor that supports direct code execution, integrated console, builtin visual debugger, package manager and more.

Smart editing for R scripts

Visual debugger

Package management

Hice builtin help
